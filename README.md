# 🚀 Terraform: Create Azure Resource Group

This project demonstrates how to create an Azure Resource Group (RG) using Terraform in a simple and reusable way.

---

## 📌 Prerequisites

Before you begin, make sure you have:

- Terraform installed (>= 1.0)
- Azure CLI installed
- An active Azure subscription

---
